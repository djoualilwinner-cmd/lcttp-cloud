# LCTTP Cloud

## Overview
LCTTP Cloud is a secure cloud platform for managing road studies, technical projects, contracts, and engineering workflows.

The platform is being developed using:

- Supabase
- FlutterFlow
- PostgreSQL
- GitHub
- OpenAI

---

## Vision

Build the leading engineering cloud platform for technical offices and public works across Algeria and beyond.
