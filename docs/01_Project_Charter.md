# 1. Project Overview

LCTTP Cloud is an enterprise SaaS platform designed for construction companies, engineering consultants, project owners, and developers in the UAE and GCC.

The platform provides a complete cloud solution to manage projects, contracts, engineering documents, finance, procurement, HR, quality, safety, and AI-powered decision support.
